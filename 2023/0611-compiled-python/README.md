# Compiled Python
Code from https://youtu.be/umLZphwA-dw.

## Setup

```sh
mamba env create -f environment.yml
```

## Usage
To run all of the benchmarks, run
```sh
./benchmark
```

To generate the Julia set visualization, run
```sh
python juliaset.py
```

To approximate the pi with taichi, run,
```sh
python taichi_pi.py
```
